Fixes font icon's not showing in TinyMCE with TinyMCE Advanced and Beaver Builder.

== Description ==
Restores correct font types to the font icons in TinyMCE when using TinyMCE Advanced with the Beaver Builder page builder plugin.

This plugin will not fix or help with missing buttons/icons added by other plugins.

== Installation ==
= The easy way: =
1. To install this plugin, click on "Add New" on the plugins page in your WordPress dashboard.
1. Search for "R3DF Beaver Builder", click install when it's found.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure the plugin in the settings page.

= The hard way: =
1. Download the latest r3df-beaver-builder-tinymce-advanced-icon-fix.zip from wordpress.org
1. Upload the zip file to the `/wp-content/plugins/` folder on your web server
1. Uncompress the zip file and delete it after it's uncompressed
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in the settings page.
